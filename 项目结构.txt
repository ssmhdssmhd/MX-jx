【核心数据流向】

  1. 用户请求 → index.php
     └─ 参数校验（URL 非空/格式/协议）
        └─ M3U8 直链检测 → 是 → 直接返回 JSON（M3U8-Direct-Output）
        └─ 否 → 进入解析流程（Strategy + Requester）

  2. Strategy 决策引擎
     └─ 匹配平台规则 → 获取优先接口
        └─ 聚合所有可用 API（api.php + ZJK.txt）
           └─ 传给 Requester

  3. Requester 并发请求引擎
     └─ curl_multi 并发请求所有 API
        └─ 收集 HTTP 200 + 非空响应
           └─ Strategy 选最优（平台优先 → 总接口最快 → 全局最快）
              └─ 代理池轮换 + 速率限制（noad.php 配置）
                 └─ 统一响应格式（JSON）

  4. NoAd 模式（mode=noad 或 auto 且启用）
     └─ NoAdParser
        └─ 资源站匹配清洗
           └─ M3U8 片段广告关键词过滤
              └─ 自定义算法应用（algorithms/）
                 └─ SQLite 记录统计

【admin.php 后台功能总览（18 个 Tab）】

  📊 仪表盘              ── PHP 版本/cURL/API 数量/环境检查/操作日志
  📈 NoAd 数据统计       ── 近 7 天柱图/广告移除数/缓存命中率/访问日志
  🔌 去广告解析源        ── NoAd 解析接口 CRUD（名称/URL/超时/启用状态）
  🚫 广告规则库          ── 自定义广告关键词（优先级/启用/命中次数）
  ⚙️ NoAd 设置          ── 缓存 TTL/超时/并发上限/关键词阈值/代理/速率限制
  📺 M3U8 解析           ── 输入远程 M3U8 URL，解析片段并标记广告
  🎬 资源站点            ── dytt/西瓜/如意/爱奇艺/腾讯/优酷/芒果 等规则管理
  📄 解析日志            ── M3U8 解析操作日志（时间/站点/片段数/结果）
  💽 缓存管理            ── 查看缓存目录/设置有效期/一键清理
  📡 API 线路            ── 管理所有解析接口（添加/删除/测试）
  🎯 平台规则            ── 视频平台 → 优先解析接口的映射规则
  🎛️ 系统开关            ── 总接口/缓存/M3U8 直链/自定义接口 等开关
  📝 自定义接口          ── ZJK.txt 在线编辑
  🧪 接口测试            ── 在线并发测试所有接口，展示响应时间和有效性
  🧩 自定义算法          ── 算法列表/启用禁用/重新扫描/本地测试/开发模板
  💾 备份日志            ── 一键备份所有配置文件，生成时间戳压缩包
  🔐 修改密码            ── 管理员密码修改（MD5 加密存储）
  ⚙️ 后台设置            ── 端口限制/IP白名单/会话有效期/文件权限等安全配置
