### 1.2 核心调用链（Legacy 模式）

```
index.php
  │
  ├─ require config/api.php        → getApiConfig()
  ├─ require config/platform.php   → getPlatformConfig()
  ├─ require config/switch.php     → getSwitchConfig()
  ├─ require core/strategy.php     → Strategy 类
  ├─ require core/requester.php    → Requester 类
  └─ require handlers/M3U8Handler.php → M3U8Handler 类

调用顺序：
  1. M3U8Handler::isM3U8Url($url)
     └─ 是 → M3U8Handler::generateM3U8Response() → exit

  2. new Strategy($apiConfig, $platformConfig, $switchConfig)
     ├─ getPriorityApi($url)
     └─ getAllApiConfigs()

  3. new Requester($strategy)
     └─ concurrentRequest($url)
         ├─ 构建所有 API 的 cURL 句柄
         ├─ curl_multi_init()
         ├─ curl_multi_add_handle() × N
         ├─ curl_multi_exec() 循环
         ├─ 收集有效响应（HTTP 200 + 非空）
         └─ $strategy->selectBestResponse($responses)

  4. 输出 JSON → exit
```

### 2.2 核心调用链（MD5 深度分析模式）

```
q.php
  │
  ├─ require config/switch.php → 检查 ad_remove_enabled / noad_enabled
  ├─ require algorithms/md5_pattern_cleaner.php → MD5PatternCleaner 类
  └─ new MD5PatternCleaner($noadCfg)
      │
      └─ deepAnalyze($m3u8Content, $videoUrl, $maxSegments)
          │
          ├─ 1. parseM3U8($m3u8Content)
          │   └─ 解析片段列表、时长、URL、DISCONTINUITY
          │
          ├─ 2. 启发式预检测（fast 模式）
          │   └─ URL 特征 + 时长异常 → 预标记
          │
          ├─ 3. 智能抽样（fast 模式）
          │   └─ 片头 + 片尾 + 中间均匀抽样
          │
          ├─ 4. 双层并发下载 + MD5 计算
          │   ├─ pcntl_fork 多进程（第一层）
          │   │   └─ curl_multi 并发下载（第二层）
          │   │       └─ md5_init / md5_update / md5_final
          │   └─ 收集所有片段的 MD5 指纹 + 文件大小
          │
          ├─ 5. 算法链应用
          │   ├─ MD5 指纹匹配 → MD5FingerprintDB::query()
          │   ├─ 如意算法 → ruyi_pattern_cleaner.php
          │   ├─ 模式特征 → pattern_feature_cleaner.php + PatternFeatureDB
          │   └─ 序列跳跃 → sequence_jump_cleaner.php
          │
          ├─ 6. 广告插播合并
          │   └─ 连续广告片段 → commercial_breaks
          │
          ├─ 7. 特征码同步（sync=1）
          │   └─ MD5FingerprintDB::syncAdSignatures()
          │       ├─ 广告特征码 → 存入广告库
          │       └─ 正片特征码 → 存入白名单
          │
          ├─ 8. 构建纯净 M3U8
          │   └─ buildCleanM3U8ByIndex()
          │
          └─ 9. 缓存保存
              └─ cache/noad_md5/{md5}.json + {md5}.m3u8
```

## 📌 关键技术点

### 解析流程核心技术
- **curl_multi 并发**：同时向多条 API 线路发起请求，取最快有效响应
- **三级选优策略**：平台优先 → 总接口最快 → 全局最快
- **M3U8 直链快速通道**：直接识别 m3u8 链接，跳过解析流程

### 去广告流程核心技术
- **双层并发架构**：pcntl 多进程 + curl_multi，速度提升 8-15 倍
- **多算法融合**：MD5 指纹 + 如意算法 + 模式特征 + 序列跳跃，互补提升准确率
- **特征码自学习**：分析结果自动存入指纹库，越用越准
- **智能抽样**：仅分析约 25% 片段，速度提升 4 倍，准确率损失极小
- **启发式预检测**：URL 强特征 + 时长异常，免下载直接判定
