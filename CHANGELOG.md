# 更新日志

## [4.5.0] - 2026-06-25

### ✨ 新增功能

- **MD5 深度分析增强**
  - 新增 Master Playlist 自动递归解析，支持多分辨率 m3u8 索引文件
  - 新增启发式广告检测（6大维度：URL关键词、时长异常、比特率异常、位置特征、众数偏离、极短片段）
  - 新增广告集群聚类分析，识别连续广告片段并自动扩展
  - 新增手动标记功能，支持人工修正广告/正片判断
  - 新增两阶段深度分析流程：先解析M3U8，再逐批分析TS片段
  - 新增 M3U8 诊断验证工具
  - 新增3种扫描模式：快速（前30段）、标准（全部）、慢速（稳定）
  - 新增广告与插播合并分析：识别 DISCONTINUITY 插播点，区分广告插播和场景切换
  - 新增 TS 特征码提取：MD5指纹、大小分布、文件名模式，支持导出 JSON 供二次开发
  - 新增广告插播段合并显示：按集群分组展示广告片段，直观显示插播位置

### 🚀 性能优化

- M3U8 下载超时从30s缩短到15s，重试从3次减少到2次
- 连接超时从8s缩短到5s
- 前端请求支持90s超时 + 自动重试2次
- 分片批量处理，避免上游请求超时
- 新增 ajax_md5_parse_m3u8 独立接口，M3U8解析与TS分析分离

### 🔧 修复

- 修复 cURL 并发下载 WRITEFUNCTION 闭包引用覆盖问题（PHP经典陷阱）
- 修复 CURLOPT_RETURNTRANSFER 与 CURLOPT_WRITEFUNCTION 冲突导致下载失败
- 修复并发下载关联数组索引越界问题
- 修复 SSL/TLS 连接错误，强制使用 TLSv1.2 + DEFAULT:!DH 密码套件
- 修复启发式检测方法未定义导致的广告检测失效问题
- 修复 parseM3U8 方法从 private 改为 public
- 修复诊断AJAX返回非JSON内容的问题
- 修复 upstream request timeout 超时问题

### 📝 其他

- 新增 deep_analysis.php 命令行深度分析工具
- 完善MD5指纹数据库自动学习与记录
- 支持手动标记结果写入指纹库
