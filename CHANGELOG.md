## [4.5.2] - 2026-06-25

### 🚀 性能优化

- **新增多进程下载分析（真正的并行处理）**
  - 使用 pcntl_fork 创建多进程，每个进程独立下载一组 TS 片段
  - 比 curl_multi 并发模式更快，充分利用多核 CPU
  - 支持 1-16 进程配置，默认 4 进程
  - 支持自动检测 CPU 核心数（设置为 0 时自动检测）
  - 自动降级：pcntl 不可用时自动回退到 curl_multi 并发模式

### ✨ 功能增强

- 后台配置新增「多进程数」参数：
  - `-1` = 禁用多进程（只用 curl_multi）
  - `0` = 自动检测 CPU 核心数
  - `1-16` = 指定进程数
- 降级机制完善：fork 失败时自动回退，确保稳定性

## [4.5.1] - 2026-06-25

### 🚀 性能优化

- **MD5 深度分析速度大幅提升**
  - 增大批次：标准模式每批从 10 段提升到 30 段（3 倍）
  - 新增启发式预检测：URL 强特征 + 时长严重异常的片段免下载直接判定，节省 30%-60% 下载时间
  - 新增智能抽样模式：片头 + 片尾 + 中间均匀抽样，仅分析约 25% 片段，速度提升 4 倍

### 🐛 Bug 修复

- 修复 `upstream request timeout` 超时错误：
  - 前端请求超时从 90s 延长到 180s
  - 重试次数从 2 次增加到 3 次
  - PHP 内存限制从 256M 提升到 512M
  - Socket 超时设置为 120s
- 修复大文件（600+ 段）分析过慢问题：通过抽样 + 预检测组合优化

### ✨ 功能增强

- 扫描模式从 3 种增加到 4 种：
  - ⚡ 极速（前 60 段·每批 20 段）
  - 🎯 智能抽样（头尾 + 中间·约 25% 片段）
  - 📊 标准（全部·每批 30 段·均衡）
  - 🐢 稳定（全部·每批 15 段·低并发）

## [4.5.0] - 2026-06-25

### ✨ 新增功能

- **MD5 深度分析增强**
  - 新增 Master Playlist 自动递归解析，支持多分辨率 m3u8 索引文件
  - 新增启发式广告检测（6大维度：URL关键词、时长异常、比特率异常、位置特征、众数偏离、极短片段）
  - 新增广告集群聚类分析，识别连续广告片段并自动扩展
  - 新增手动标记功能，支持人工修正广告/正片判断
  - 新增两阶段深度分析流程：先解析M3U8，再逐批分析TS片段
  - 新增 M3U8 诊断验证工具
  - 新增3种扫描模式：快速（前30段）、标准（全部）、慢速（稳定）
  - 新增广告与插播合并分析：识别 DISCONTINUITY 插播点，区分广告插播和场景切换
  - 新增 TS 特征码提取：MD5指纹、大小分布、文件名模式，支持导出 JSON 供二次开发
  - 新增广告插播段合并显示：按集群分组展示广告片段，直观显示插播位置

### 🚀 性能优化

- M3U8 下载超时从30s缩短到15s，重试从3次减少到2次
- 连接超时从8s缩短到5s
- 前端请求支持90s超时 + 自动重试2次
- 分片批量处理，避免上游请求超时
- 新增 ajax_md5_parse_m3u8 独立接口，M3U8解析与TS分析分离

### 🔧 修复

- 修复 cURL 并发下载 WRITEFUNCTION 闭包引用覆盖问题（PHP经典陷阱）
- 修复 CURLOPT_RETURNTRANSFER 与 CURLOPT_WRITEFUNCTION 冲突导致下载失败
- 修复并发下载关联数组索引越界问题
- 修复 SSL/TLS 连接错误，强制使用 TLSv1.2 + DEFAULT:!DH 密码套件
- 修复启发式检测方法未定义导致的广告检测失效问题
- 修复 parseM3U8 方法从 private 改为 public
- 修复诊断AJAX返回非JSON内容的问题
- 修复 upstream request timeout 超时问题

### 📝 其他

- 新增 deep_analysis.php 命令行深度分析工具
- 完善MD5指纹数据库自动学习与记录
- 支持手动标记结果写入指纹库
